Questions

Frequently asked questions

What's the difference between Starter, Standard, and Pro?
Starter is a 2-tab Google Sheets file — donor list and gift log. Standard is a 5-tab system that adds Moves Management, Grant Tracker, and a Board Dashboard. Pro is a full web app with multi-user access, real-time cloud sync, and board report charts. All three include the same core donor tracking; they differ in depth and delivery format.
Do I need to know Excel or Google Sheets to use this?
No. The formulas and calculations are already built in. You just type into cells — the system does the math. Each file includes a plain-English Start Here guide that explains every column and every term.
Can multiple people use the same file?
For Starter and Standard: you can share the Google Sheet with teammates the same way you'd share any Google file. For Pro: multi-user access is built in — your whole team logs into the same shared database simultaneously with their own accounts.
Is my data safe in DonorDesk Pro?
Yes. Pro is powered by Google Firebase — the same infrastructure used by major enterprise applications. Your organization's data is completely private and separate from every other organization. Auto sign-out activates after 60 minutes of inactivity.
What happens to my data if I cancel Pro?
Before cancelling, go to Settings → Export Backup to download a complete JSON file of all your donors, gifts, moves, and grants. Your data is always yours. We recommend exporting a backup monthly regardless.
Can I upgrade from Starter or Standard to Pro later?
Yes. Pro is a separate product — you can start it anytime. Your existing spreadsheet data can be manually entered into Pro, or you can run both in parallel during a transition period.
Do I need a Gmail address to use Pro?
No. Any email address works — your work email, a personal email, whatever you prefer. Google Sign-In is available as an option but not required.
How do I get support if I have a question?
Email donordesk2@gmail.com — responses within 3 business days. DonorDesk Pro also has a built-in Help & FAQ section with 38 answered questions, searchable in real time.
